## Aim: 
Comparative Analysis of different types of Prompting patterns and explain with Various Test Scenarios

Experiment:
Test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. 
Analyze the quality, accuracy, and depth of the generated responses.


## Algorithm:

Algorithm
Step 1: Define Objective
Identify the goal: Compare performance of different prompting techniques
Focus on response quality, reasoning ability, and accuracy
Step 2: Select Prompting Patterns

Choose commonly used prompting techniques:

Zero-shot Prompting → No examples given
Few-shot Prompting → Few examples included
Chain-of-Thought (CoT) → Step-by-step reasoning
Role-based Prompting → Assigning a role (e.g., “expert”)
Structured Prompting → Fixed output format
Step 3: Design Test Scenarios

Create diverse scenarios to test AI capability:

General Knowledge
Technical Question Answering
Logical Reasoning
Creative Writing
Data Interpretation
Step 4: Create Prompt Variants

For each scenario:

Broad Prompt → Vague instruction
Refined Prompt → Clear, structured instruction
Step 5: Execute Experiment
Run all prompt types on selected AI models
Collect responses for each scenario
Repeat multiple times to ensure consistency
Step 6: Evaluate Responses

Use the following metrics:

Accuracy
Relevance
Clarity
Depth
Creativity
Logical reasoning

 Score each from 1 to 10

Step 7: Record Results
Store scores in tabular format
Calculate average score per prompting pattern
Step 8: Compare & Analyze
Identify best-performing pattern
Compare broad vs refined prompts
Analyze consistency and reliability
Step 9: Generate Insights
Determine which prompting technique works best for each scenario
Identify weaknesses of each pattern
Step 10: Conclusion
Summarize overall best prompting strategy
Provide recommendations

## Key Observations
Refined prompts always outperform broad prompts
Chain-of-Thought (CoT) gives highest accuracy in reasoning tasks
Few-shot prompting improves learning with examples
Role-based prompting improves clarity and tone
Structured prompting works best for data-based tasks


## Result

Best Overall Pattern: Chain-of-Thought (CoT)
Best for Beginners: Few-shot Prompting
Best for Data Tasks: Structured Prompting

## Conclusion

The experiment proves that:

Prompt design directly impacts AI performance
Structured and reasoning-based prompts produce better results
Combining techniques (CoT + Few-shot) gives optimal performance
